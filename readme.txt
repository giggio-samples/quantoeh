QuantoEh é uma calculadora social para ser utilizada no twitter rodando na nuvem com Microsoft Azure. QuantoEh é uma calculadora para ser utilizada no twitter. Com ela você envia uma mensgaem do tipo: "@quantoeh 2+2" e então receberá um RT com a resposta. Desenvolvido com: ASP.NET MVC Azure Worker Role C# 4.0 Boo 
Desenvolvida por: 
Felipe Rodrigues @felipero 
Giovanni Bassi @giovannibassi 
Vinicius Quaiato @vquaiato

É um projeto desenvolvido para se divertir. É uma prova de conceito de que algumas tecnologias funcionam.

Alguns arquivos que podem lhe interessar:
- rodando.txt => explica como rodar o quanto é na sua máquina, fora da nuvem
- linhadecomando.txt => explica como compilar e opcionalmente rodar testes via linha de comando
- arquivosquefaltam.cmd => alguns arquivos não foram versionados por conterem segredos (como a senha do twitter do quanto é, do azure, etc). Você pode rodar este arquivo para criar arquivos de modelo para poder rodar o projeto.
